🛠 About This Project

This application performs sentiment analysis on real movie reviews fetched from TMDb using a RoBERTa sentiment model hosted on Hugging Face.

Search for a movie, select one of the matching results, and instantly visualize audience sentiment together with individual review classifications.

Developed by Anshika Gupta
