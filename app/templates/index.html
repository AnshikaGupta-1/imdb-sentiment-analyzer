🛠 About This Project

This app fetches real audience reviews for any movie from TMDb, then classifies each one using cardiffnlp/twitter-roberta-base-sentiment-latest, a RoBERTa-based transformer fine-tuned for 3-class sentiment (positive / neutral / negative), served via the Hugging Face Inference API.

Reviews are classified concurrently and the results are aggregated into a positive/neutral/negative breakdown plus a single confidence-weighted overall score, so you get both the granular per-review view and a quick at-a-glance read on audience reception.

Developed by Anshika Gupta
